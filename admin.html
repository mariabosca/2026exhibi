How to create your token (once a year)
  1. Open github.com/settings/personal-access-tokens/new (sign in as mariabosca).
  2. Token name: gallery. Expiration: 1 year (the maximum).
  3. Repository access: Only select repositories → choose 2026exhibi.
  4. Permissions → Repository permissions → Contents: Read and write. Leave everything else alone.
  5. Click Generate token, copy it (starts with github_pat_) and paste it above. It stays saved in this browser until you sign out.
